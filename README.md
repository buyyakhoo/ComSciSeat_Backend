# ComSciSeat Backend
Backend computer laboratory table reservation web application for Computer Science students

## Related Repository
- Frontend: https://github.com/buyyakhoo/ComSciSeat_Frontend

## Tech Stack
- TypeScript
- Hono
- Prisma
- Resend
- OAuth2Client

## External Services
- Google OAuth 2.0 - use OAuth2Client Library
- Email Service API - use Resend library

## Create a Project, Developing, and Building
```
npm install
npx prisma init
npx prisma pull
npx prisma generate
npm run dev
```

```
open http://localhost:3000
```
